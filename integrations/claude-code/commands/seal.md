# View SEAL Patterns

Display all learned patterns from the SEAL engine with current confidence scores.

## What to Show:

Read [.vibecoding/PROJECT-STATE.md](../.vibecoding/PROJECT-STATE.md) and display:

1. **Section 5: SEAL LEARNED PATTERNS** (complete)
   - All pattern categories (5.1 through 5.10)
   - Confidence scores for each
   - Number of tasks patterns learned from

2. **Summary**:
   - Total tasks completed
   - Success rate (rating ≥ 4)
   - Strongest patterns (confidence > 0.70)
   - Weakest patterns (confidence < 0.40)

3. **Recommendations**:
   - Which patterns need more data
   - Which patterns are ready for automatic application (confidence ≥ 0.70)

## Usage:

User types: `/seal`
Result: Complete view of all learned patterns and their maturity levels
